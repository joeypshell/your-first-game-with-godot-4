# Your First 2D Game With Godot 4

Godot 4 project based on the GDQuest tutorial starter.

## Open locally

1. Install Godot 4.7 or newer.
2. Clone this repository.
3. In Godot's Project Manager, choose **Import** and select `project.godot`.

## Source control notes

- Commit project source files such as `.gd`, `.tscn`, `.tres`, assets, and `.import` metadata.
- Do not commit `.godot/`; it is local editor cache and is ignored.
- Commit and push changes from one computer before pulling on the other.
